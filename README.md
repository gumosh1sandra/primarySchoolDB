Primary School Management Database System (primarySchoolDB)


Student Name: GUMOSHABE SANDRA

This database manages student profiles, teacher information, academic subjects and student financial accounts thus creating order and easy storage.
The system is built using 6 related tables that is;
Teacher Table
Pupil Table
Subject Table
Class Table
Enrollment Table
Fees Table




How to run the Project
Requirements
 Microsoft SQL Server (2019 or later recommended) and SQL Server Management Studio (SSMS)

  Clone this repository to your machine
  Open SQL Server Management Studio and connect to your database
  Open the main script file named primarySchoolDB.sql
  Execute the code blocdk by block

  OR

Open SQL Server Management Studio (SSMS).
Create a new database named PrimarySchoolDB.
Open the file PrimarySchoolDB.sql.
Execute the code to create all tables and insert sample data.
Verify data using:
SELECT * FROM Teacher;
SELECT * FROM Pupil;
SELECT * FROM Subject;
SELECT * FROM Class;
SELECT * FROM Enrollment;
SELECT * FROM Exam;
SELECT * FROM Result;
SELECT * FROM Fees;
