# primarySchoolDB
PRIMARY SCHOOL MANAGEMENT SYSTEM
This database manages student profiles, teacher information, academic subjects and fees bills thus creating order and easy storage.
The system is built using 6 related tables that is;
 Teacher
 Pupil
Subject
Class
Enrolment
Fees
How to run the database
Open SQL Server Management Studio (SSMS).
Create a new database named PrimarySchoolDB.
Open the file PrimarySchoolDB.sql.
Execute the script to create all tables and insert sample data.
Verify data using:
SELECT * FROM Teacher;
SELECT * FROM Pupil;
SELECT * FROM Subject;
